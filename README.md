Swag Labs UI Automation and API Automation– Playwright + TypeScript

This project contains an automated test for the Swag Labs demo application and API using Playwright with TypeScript.

Prerequisites: Node.js (v18 or later recommended) and npm

Run all tests: npx playwright test
Run tests in UI mode: npx playwright test --ui
Run a specific test file: npx playwright test tests/swaglab.spec.ts 

Playwright automatically generates an HTML report after execution.
Open the report: npx playwright show-report or can find the report in \test-results folder
